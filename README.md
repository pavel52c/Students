# Students

## Установка пакетов:

Из корневой папки:

`cd node && npm i`

`cd vite-project && npm i`

## Настройка работы с БД:

В проекте node необходимо внести свои данные в файл 

`ormconfig.ts`


## Запуск проекта:

Из корневой папки:

`cd node && npm run start`

`cd vite-project && npm run dev`
